# visa-backoffice
PROJET VISA (MR NAINA)

2. Architecture Technique
Back-office : SpringBoot (Java).
Front-office : Framework Ewan
Sécurité : Authentification JWT. Les rôles et les permissions (actions) sont embarqués dans le Token pour minimiser les accès base de données.
Optimisation : Mise en session des tables de référence (Types de visa, Statuts) pour un accès instantané côté client.
